# MAE 106 Autonomous Pneumatic Robot — UCI

## Overview
Autonomous robot designed and built for the UCI MAE 106 (Mechanical Systems Laboratory) final competition: *Attack on the Droid Anthill*. Built by a team of three, with roles split across Mechanical, Electrical, and Controls.

The robot uses a single pneumatic cylinder for propulsion, a servo motor for steering, and an Arduino microcontroller to navigate a trench course autonomously — no human input once the race starts.

## My Role: Mechanical Coordinator
- Designed full chassis assembly in SolidWorks (frame, wheel mounts, steering linkage)
- Selected materials: plywood chassis decks, FDM-printed steering and mount components
- Fabricated components using band saw, drill press, laser cutter, and Bambu X1C 3D printer (PLA)
- Ensured compliance with competition constraints: 12" diameter envelope, Air Tank tire centered at 16" height
- Coordinated mechanical interfaces with Electrical and Controls subteams

## System Architecture
| Subsystem | Component |
|---|---|
| Propulsion | Pneumatic cylinder (compressed air, max 40 PSI) |
| Steering | Servo motor + custom steering mechanism |
| Controller | Arduino microcontroller |
| Sensors | Magnetometer (bearing), Limit Switch (odometry) |
| Structure | Plywood chassis + FDM 3D printed parts |

## Competition
- Course: Outdoor trench navigation, Engineering Gateway courtyard, UCI
- Scoring: Distance-based, up to 150 pts for trench crossing
- Robot must actuate fully autonomously within a 60-second race window
